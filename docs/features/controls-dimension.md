## Controls dimension <!-- {docsify-ignore-all} -->

You could change the dimension of controls/menu by changing the `font-size` of className `.video-js`.

```css
@media (min-width: 1440px) {
  .video-js {
    font-size: 18px;
  }
}

@media (min-width: 1680px) {
  .video-js {
    font-size: 20px;
  }
}
```
